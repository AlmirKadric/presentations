# Observium - Mage

What is it and why do we need it

  1. Observium is a data collection and graphing tool. The mage application allows us to tie this to a running game and get real-time statistics.
  2. It allows us to quickly see changes and trends on our systems over a period of time.
  3. It helps identify potential problems before they occur as well as keep track of valuable KPI's.

How does it work

  1. We poll each game on each system periodically and store this data into a Round Robin Database.
  2. A configuration file defines which data should / should not be collected and where to collect it from.
  3. The configuration file also defines individual graphs, their paths and what data to present.
